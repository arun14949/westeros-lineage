## Instructions

1. For each character, prepend the **style prefix** to the prompt column
2. Generate a square portrait image (512x512 or 1024x1024)
3. Save as PNG with the exact filename from the table
4. Place all images in `public/images/` folder
5. The app will automatically pick them up (image paths in `src/data.ts` will be updated to `/images/{filename}`)
